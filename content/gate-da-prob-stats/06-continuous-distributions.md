## Q501

[MCQ]

$X \sim U(2, 8)$. What is $E[X]$?

- **(A)** 2
- **(B)** 3
- **(C)** 5
- **(D)** 6

**Answer:** C

## Q502

[NAT]

$X \sim U(0, 10)$. What is $\text{Var}(X)$?

$$\text{Var}(X) = \frac{(b-a)^2}{12} = \frac{100}{12} = \frac{25}{3} \approx 8.33$$

**Answer:** 8.33

## Q503

[MCQ]

$X \sim U(2, 10)$. What is $P(4 < X < 7)$?

- **(A)** $\dfrac{3}{8}$
- **(B)** $\dfrac{5}{8}$
- **(C)** $\dfrac{3}{10}$
- **(D)** $\dfrac{7}{10}$

**Answer:** A

## Q504

[MSQ]

Which of the following are true for $X \sim \text{Exponential}(\lambda)$? (Select all that apply)

- **(A)** $E[X] = \dfrac{1}{\lambda}$
- **(B)** $\text{Var}(X) = \dfrac{1}{\lambda^2}$
- **(C)** $P(X > t) = e^{-\lambda t}$ for $t > 0$
- **(D)** $X$ takes values on $(-\infty, \infty)$

**Answer:** A, B, C

## Q506

$X \sim \text{Exponential}(\lambda = 0.5)$. Given that $X > 3$, what is $P(X > 5)$?

- **(A)** $e^{-2.5}$
- **(B)** $e^{-1}$
- **(C)** $e^{-0.5}$
- **(D)** $e^{-2}$

## Q507

[MCQ]

$X \sim N(\mu, \sigma^2)$. The Z-score transformation is $Z = \dfrac{X - \mu}{\sigma}$. If $X \sim N(10, 25)$, what is $P(X < 15)$?

- **(A)** $P(Z < 0)$
- **(B)** $P(Z < 1)$
- **(C)** $P(Z < 2)$
- **(D)** $P(Z < 5)$

**Answer:** B

## Q508

[NAT]

For a standard normal distribution, $P(-1 < Z < 1) \approx 0.6827$. If $X \sim N(50, 100)$, what is $P(40 < X < 60)$? (Here $\sigma = 10$.)

**Answer:** 0.6827
